# 완료 TODO 아카이브

> 지지난주 이전 완료 항목의 히스토리. TODO.md 의 `완료 (이번 주 + 지난 주)` 섹션에서 매주 월요일 (또는 정리 시) 이관됨.
> 상세 파일은 `todos/TODO-NNN-{slug}.md` 에 그대로 보존 — 본 파일에는 인덱스 행만 누적.

## 운영 메모

- **이관 기준**: 완료일 기준 지지난주 이전 항목 (예: 오늘이 5/14 목요일이면 ~5/3 이전 완료 항목)
- **이관 시점**: 매주 월요일 또는 사용자 정리 요청 시
- **이관 절차**:
  1. TODO.md `완료 (이번 주 + 지난 주)` 섹션에서 이관 대상 행 식별
  2. 본 파일의 해당 월 섹션에 행 추가 (월 섹션 없으면 신설)
  3. TODO.md 에서 해당 행 제거
- **상세 파일**: `todos/TODO-NNN-{slug}.md` 는 이관과 무관하게 보존
- **컬럼 형식**: `ID | 제목 | 등록일 | 시작일 | 완료일 | 상세` (TODO.md 완료 섹션과 동일)
- **정렬**: 각 월 섹션 내에서 완료일 역순 (최신이 위)
