### 기획 시 결정 필요 항목

- **알림 대상 채널**: 신규 채널? 기존 채널 활용? (운영 / 사업 / 데이터 어느 쪽?)
- **쿠폰 등록 트리거**: 어느 시점 — 쿠프마케팅 등록 직후 / 헬로우봇 시스템에 동기화된 시점 / 사용자 첫 사용 시점?
- **쿠폰 등록 알림 항목**: 상품명·가격·등록자·등록 시각 / 추가 메타데이터 (재고·만료일 등)?
- **일별 통계 항목**: 등록 건수 / 사용 건수 / 매출 / 신규 사용자 / 환불·취소 / 채널별 분포?
- **데이터 소스**: 서버 DB(PostgreSQL) / BigQuery / 쿠프마케팅 API 어느 쪽?
- **기술 스택**: 서버 cron 잡 / Airflow DAG / 서버 이벤트 핸들러 + Slack Webhook?
- **에러/장애 처리**: 알림 발송 실패 시 재시도·알림 누락 통지 방법
