# 메가존소프트 조치 가이드 (외부 산출물)

> 파트너사 메가존소프트(김종현 대표님)로부터 수신한 GCP Gemini 이상비용 조치 가이드 + 스크립트.

## 추가 안내

이 폴더에 다음을 추가:

- **가이드 문서**: PDF / docx / md 등 원본 형식 그대로
- **스크립트 파일**: shell / Python / SQL 등 실행 환경에 맞게
- **첨부 자료**: 스크린샷·차트·로그 샘플 등

## 사용 흐름

1. 사용자가 본 폴더에 가이드 + 스크립트 파일 추가
2. 코디네이터(또는 `/dev-infra` 위임)가 가이드 분석 → `../../tasks.md` 의 "조치 실행" 섹션을 항목별 과업으로 분해
3. 스크립트 실행 시 본 폴더 경로에서 진행 (실행 로그도 본 폴더에 누적 권장)
