## 전 에이전트 수렴 — 확정 결함 (독립 다수결)

### ★ C1 — 추천/전이의 모집단(n=23,049)이 잘못됨 (8개 에이전트 수렴: 01·02·03·04·05·07·09·10)
- R2-6/R2-8/U-1 전체가 올라탄 "재방문 2회차 23,049명"은 `has_2nd=1 AND second_sameday=0`(전역 2번째 구매가 다른 날) — **90일 상한 없음**(5,492명, 23.8%가 90일 초과)이고, **당일 업셀 후 재방문(3번째+)한 20,178명(진짜 재방문자의 53.5%)을 누락**.
- 계약상 rev90 재방문 모집단 = 37,735명(38.4%). "재방문"이라는 한 단어가 38.4%(헤드라인)와 23.4%(R2-2)를 동시에 가리킴.
- 결과: 전이/매트릭스가 "재방문 시 무엇을 사는가"를 대표하지 못함. **재집계(re-export) 필요**.

### ★ C2 — "반복=콘텐츠 특성(타로>사주)" 헤드라인이 metric 오염 (06·09)
- 타로>사주 격차는 **당일 현상**. rev90에서 당일을 제거한 순수 재방문 리텐션에선 **사주가 4.0pp 우위**. Simpson's paradox(가격tier 내 타로≥사주이나 집계는 사주 우위).

### ★ C3 — "연애=universal default 2nd topic"는 base-rate 가공물 (09, 보강 05·07)
- 연애 = 전체 재방문 2번째 구매의 73.6%. 1st topic을 빼면 디폴트 붕괴. 연애 자기-prior 대비 lift ≤1.21, **8셀 중 3셀에서 lift<1**. 매트릭스가 lift(증분)가 아닌 confidence(인기)를 최적화. "lift 2.01"은 topic+ctype 결합 분모로 부풀려진 값(정직한 topic-prior lift=0.86<1).

### ★ C4 — R2-6 전이 셀 수치 stale/미재현 (05·10 독립 수렴)
- love→love 79.4%→실제 83.8%, family→family 42.9%→53.7%, 타로→사주 17.3%→26.5%. rev2 전이 스크립트 미보존, rev1/ever와 rev2/revisit 사이 손편집 흔적. **단, U-1 매트릭스는 소수점까지 정확 재현** → U-1은 신선·정확, R2-6만 오염.

### M1 — 불확실성 규율 전무 (07)
- baseline 어디에도 CI·n·유의성 없음. n=188~573 셀이 ±0.3pp 헤드라인과 동일 활자. "<25 타로 lift 2.01 최강"은 ~300셀 중 극값(winner's curse). "저연령일수록 lift↑"는 비단조(2.01→1.63→1.60→1.66), 45+ 타로 실제 1.66(>1)인데 셀 suppress로 "고연령 약함" 서사 유지.

### M2 — 데이터 품질: age 컬럼 오염 (01·03)
- `age` min −20,048,780 / max 2,010(출생연도). "98% 채움"은 사실이나 채움≠유효. R2-7 연령 라이프사이클이 미검증 값 위에 구축.

### M3 — 채널 정의 불일치 (08)
- β 분류기 = `pay_%`(83원 마이크로 포함) vs A 코호트 = `pay_for_%`. 채널 사이징을 pay_for_ 기준으로 재실행한 적 없음. 웹 활동이 무료/sub-750였던 웹획득 유저가 A로 흡수(B 누수, 단방향).

## 전원 검증 통과 (반박 없음)
- 헤드라인 이진 % 전부 정확 재현: N=98,392 / rev90 38.35% / sameday 50.04% / any90 67.88% / ever 73.47%.
- `pay_for_%` + revenue_krw>0 필터 정확, 마이크로거래 배제 실재(f_rev 최소 150, 전부 150 배수).
- `c_any90 = c_rev90 + c_sameday` 파티션 행단위 무결, 30/60/90 단조성 무위반.
- **U-1 통합 매트릭스 8개 A셀 소수점까지 재현** (앞선 세션의 셀 교정이 옳았음을 독립 확인).
