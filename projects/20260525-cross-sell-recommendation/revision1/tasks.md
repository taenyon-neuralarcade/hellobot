## Phase 2 — 세그먼트 분류 + Profiling (1.5일, 6/3 화 ~ 6/4 수 오전)

> 가이드 3단계 (Segmentation) + 4단계 (Profiling)

### 2.1 세그먼트 축 검증 + 라벨링

- [ ] **후보 축 단독 스크리닝** — 연령 5세 / 성별 / 첫 결제 토픽 / 가격대 / 페르소나 각각 재구매율 차이 (chi-square + effect size)
- [ ] **유의미 축 선별** — 차이가 큰 축 2~3개 채택
- [ ] **세그먼트 구성** — 선별 축 곱 + S1~S6 와 비교 (수렴 여부 확인)
- [ ] 셀별 인원수 분포 — 인원수 작은 셀 (예: <50명) 제외 또는 합치기 기준 결정

### 2.2 프로파일 표 작성

- [ ] **세그먼트별 핵심 지표 표** — 인원수, 평균 첫 결제가, 재구매율(30/60/90d), 두 번째까지 median 일수
- [ ] **TOP3 첫 카테고리 + TOP3 두 번째 카테고리** — 셀별
- [ ] **콘텐츠 소비 행태** — 무료 진입 횟수, 진입 토픽 분포 등 보조 시그널

### 2.3 통계 검정

- [ ] **chi-square** — 세그먼트 × 재구매 여부
- [ ] **ANOVA / Kruskal-Wallis** — 세그먼트 × 두 번째까지 일수
- [ ] **effect size** — Cramer's V, eta-squared 측정 (p-value 만 의존 X)

### 2.4 우선순위 셀 추출

- [ ] 베이스라인 대비 재구매율 ±10%p 이상 벗어난 셀 선별
- [ ] **High Performer / Underperformer / Volume Driver** 분류
- [ ] **우선 공략 셀 TOP 5~10** 픽스 → Phase 3·4 의 입력

### 2.5 산출물

- [ ] `data/segment-labeling.sql`
- [ ] `data/segment-profile.csv`
- [ ] `planning/segment-profile.md` — 프로파일 표 + 통계 검정 결과 + 우선순위 셀

## Phase 4 — 추천 SKU 매트릭스 도출 (1.5일, 6/5 목 오후 ~ 6/8 월 오전)

> 가이드 5단계 (Affinity / NBO) 의 어소시에이션 + CF 결합

### 4.1 어소시에이션 룰 계산

- [ ] **(첫 SKU → 두 번째 SKU) 페어별** support, confidence, lift
- [ ] **세그먼트별** lift 재계산 (전체 vs 세그먼트 차이 확인)
- [ ] Support 너무 작은 페어 제외 (인원수 50명 미만 등)

- [ ] (우선순위 셀 × 첫 결제 SKU) → 두 번째 결제 TOP 5 SKU + lift

- [ ] **추천 매트릭스 완성** — 본 분석의 핵심 산출물
